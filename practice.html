Practice Your Microsoft Office Skills

Choose an exercise below and test what you have learned. Practice helps you master the skills faster.

Microsoft Word Practice

  1. Create a new document and type a short paragraph.
  2. Change the font size and style.
  3. Insert a table with 3 rows and 3 columns.
  4. Insert an image into the document.
  5. Save the document as practice.docx.

Microsoft Excel Practice

  1. Create a table with student names and scores.
  2. Use the SUM formula to calculate totals.
  3. Use the AVERAGE function.
  4. Create a simple bar chart.
  5. Save the spreadsheet.

Microsoft PowerPoint Practice

  1. Create a presentation with 5 slides.
  2. Add a title slide.
  3. Insert images on two slides.
  4. Add slide transitions.
  5. Run the slideshow.

Microsoft Access Practice

  1. Create a new database.
  2. Create a table called Students.
  3. Add fields: ID, Name, Age, Course.
  4. Enter at least 5 student records.
  5. Create a query to show students older than 18.
